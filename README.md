# Microbit V2 Examples

## Prerequisites

### Hardware

* [BBC Micro:bit v2](https://microbit.org/new-microbit/)

### Software

Make sure you have the latest versions (`cargo install <tool>`) of these tools:

* `flip-link`
* `probe-run`
* `cargo-embed`

## Usage

### run application

```
cargo rb led-roulette
```

### flush application

```
cargo embed --bin led-roulette
```
